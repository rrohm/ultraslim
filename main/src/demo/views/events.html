Events

ondblclick

ondblclick event handling supports:

  • JS code in HTML elements, e.g.,:
    <span ae-dblclick="alert('Oh!')">...</span>
  • References to controller methods , e.g.:
    <span ae-dblclick="this.controller.onDblClick">...</span>

onclick

onclick event handling supports:

  • JS code on HTML elements, e.g., spans:
    <span ae-click="alert('Oh!')">...</span>
  • References to controller methods , e.g.:
    <span ae-click="this.controller.onClick">...</span>
