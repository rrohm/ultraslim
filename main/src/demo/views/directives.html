Directives & events

The ultraslim library supports directives for templating, rendering model data and for binding event handlers, e.g., in this way: 

<li ae-repeat="data" ae-click="this.controller.handle({{id}})" id="data_{{id}}">{{name}}</li>
  • {{name}}
  • {{name}} ({{address.street}})

The ultraslim library support these events:

  • ae-click
